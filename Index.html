Sobre o Projeto

Essa é uma tradução completa feita por fãs para o jogo Inazuma Eleven GO Strikers 2013, totalmente em português do Brasil! O projeto inclui menus, técnicas, falas, nomes e tudo que os fãs merecem para uma experiência 100% BR.

Ferramentas utilizadas: KUURIMU, Strikers Tools, Balwcrate, Photoshop.
